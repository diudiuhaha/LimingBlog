# Front-matter
基本属性
```
---
title:  
published:  
description:  
image:  
tags: 
category: 
slug: 
---
```

附加属性
```
//是否置顶在文章列表顶部
pinned: false

//是否为草稿
draft: false

//是否启用评论
comment:

//文章密码，设置后文章将被加密保护
password:

//密码提示，显示在密码输入框上方
passwordHint:
```


# 剧透文本
使用 `:spoiler[]` 语法隐藏文本内容，支持 Markdown 格式：
```
内容 :spoiler[被隐藏了 **哈哈**]！
```
